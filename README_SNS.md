🎉 **ChulipVideo v0.2.3 リリース！** 🎉

アプリ名称を「ChulipVideo」に一新しました！🌷
未実装だった録画の細かい機能と字幕表示が大幅に強化されています💪

**✨ 主な新機能:**
✅ **マウスポインタのオーバーレイ**: 録画時のカーソル位置・クリック波紋を再生時に再現🖱️
✅ **キー・クリック入力字幕**: 入力履歴を字幕として表示。フェードアウトや複数表示に対応⌨️
✅ **詳細設定ダイアログ**: テーマやオーバーレイの見た目をGUIでカスタマイズ可能に🎨
✅ **録画機能強化**: ウィンドウ単体キャプチャのWGC対応、録画開始までのカウントダウン表示、画質変更、FPS変更

その他、細かな不具合修正や使い勝手の向上を行いました。
