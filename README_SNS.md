# ChulipVideo v0.2.3 - SNS投稿用

## 📢 v0.2.3 アップデート！

ChulipVideoを更新しました！UI視認性と操作の安定性が大幅に向上しています✨

### 🎯 主な新機能

**ウィンドウ状態の完全復元**
- 前回のウィンドウ位置・サイズ・最大化状態を記憶
- マルチモニター環境でも正確に復元（セカンダリモニターOK）

**再生位置の記憶**
- 動画ごとに最後の再生位置を保存
- 次回起動時に続きから再生できる

**レイアウトの改善**
- 下側の操作パネルを450pxに固定
- 誤って動かせないように改良
- ウィンドウ最小サイズを800x600に設定

### 🎨 UIハイコントラスト化

ボタンの配色が鮮やかにしました！

### 🔍 ズーム操作の安定化

- ホイールズームを「レベル制」に変更 → 必ず100%に戻れるように！
- パンニング中でもキャンバス中央基準でズーム → 見たい場所がズレない！

### 💡 こんな人におすすめ

- マニュアル作成で大量のスクショが必要な方
- 動画から特定の範囲を繰り返し切り出す方

Python + Tkinter + OpenCV + Pillow
