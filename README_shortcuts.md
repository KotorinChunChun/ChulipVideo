Space: 再生/停止
Ctrl+Z: 戻す (Undo crop)
Ctrl+Y: 進む (Redo crop)
Ctrl+C: 現在のクロップをクリップボードにコピー
Left/Right: 0.1s 移動、Ctrl+Left/Right: 1s 移動
Home/End: Start/End に移動
Ctrl+Home/End: 動画の先頭/末尾へ移動

クリック（左）: クロップをフォーカス（オレンジ枠）
ドラッグ（左ボタン）: クロップの移動/角リサイズ
Ctrl+ドラッグ: 中心固定でリサイズ（反対側も連動）
右ドラッグ: 矩形移動
Shift+角ドラッグ: アスペクト比を維持してリサイズ
Alt+矢印: クロップを1ピクセル単位で移動（矩形にフォーカスがある場合）
Shift+矢印: クロップを中心に1ピクセルずつ拡大・縮小（上下で高さ、左右で幅）

中ボタン（押し込み）+ドラッグ: 画像のパン
中ボタン ダブルクリック: 拡大縮小・パンのリセット
マウスホイール: ズーム（拡大縮小）
