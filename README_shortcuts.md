Space: 再生/停止
Ctrl+Z: 戻す (Undo crop)
Ctrl+Y: 進む (Redo crop)
Ctrl+C: 現在のクロップをクリップボードにコピー
Left/Right: 0.1s 移動、Ctrl+Left/Right: 1s 移動
Home/End: Start/End に移動
Ctrl+Home/End: 動画の先頭/末尾へ移動

クリック（左）: クロップをフォーカス（オレンジ枠）
左ドラッグ: クロップの移動/角リサイズ
Alt+左ドラッグ: 10ピクセル単位でクロップの移動/角リサイズ
Ctrl+左ドラッグ: 中心固定でリサイズ（反対側も連動）
Shift+左ドラッグ: 垂直または水平方向に移動を限定
右ドラッグ: 矩形移動
Shift+角ドラッグ: アスペクト比を維持してリサイズ
Alt+矢印: クロップを1ピクセル単位で移動（Alt+Ctrl+矢印で10ピクセル移動）
Shift+矢印: クロップを中心に1ピクセルずつ拡大・縮小（Shift+Ctrl+矢印で10ピクセル拡縮）


中ボタン（押し込み）+ドラッグ: 画像のパン
中ボタン ダブルクリック: 拡大縮小・パンのリセット
マウスホイール: ズーム（拡大縮小）
