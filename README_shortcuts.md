# ショートカットキー一覧

基本的にはExcelやAutoCADの操作に準拠する。

## 動画再生関連:

- Space: 再生/停止
- Left/Right: 0.1s 移動
- Ctrl+Left/Right: 1s 移動
- Home/End: Start/End に移動
- Ctrl+Home/End: 動画の先頭/末尾へ移動
- マウス戻る/進むボタン: -1s / +1s 移動

## クロップ操作関連:

### 基本操作

- 左クリック: クロップをフォーカス
- 右クリック: 新たにクロップを作成（未実装）
- Ctrl+Z: 戻す (Undo crop)
- Ctrl+Y: 進む (Redo crop)
- Ctrl+C: 現在のクロップをクリップボードにコピー

### 枠線の操作(リサイズ)

- 左ドラッグ: リサイズ
- Alt+左ドラッグ: 10ピクセル単位でリサイズ
- Ctrl+左ドラッグ: 中心を起点として両端をリサイズ
- Shift+左ドラッグ: (角のみ)縦横アスペクト比を維持してリサイズ
- Shift+矢印: クロップを中心に1ピクセルずつ拡大・縮小
- Shift+Ctrl+矢印: クロップを中心に10ピクセルずつ拡大・縮小

### 領域の操作(移動)

- 左ドラッグ: クロップを1ピクセル単位で移動
- Alt+左ドラッグ: クロップを10ピクセル単位で移動
- Shift+左ドラッグ: 移動方向を垂直または水平方向に限定
- Alt+矢印: 指定方向へクロップを1ピクセル単位で移動
- Alt+Ctrl+矢印: 指定方向へクロップを10ピクセル単位で移動

## ビュー関連:

- マウスホイール: ビューの拡大縮小
- 中クリック+ドラッグ: ビューの移動（パンニング）
- 中ダブルクリック: ビューのリセット（拡大縮小やパンを元に戻す）
